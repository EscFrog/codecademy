Walter's Cheet Sheet

Table Tags

Tag Name Discription
<table> Table 각각의 테이블 전체를 감싸는 태그
<thead> Table Head 테이블의 제목 행을 정의하는 태그
<tbody> Table Body 테이블의 실제 데이터들이 들어갈 부분
<tr> Table Row 테이블 각각의 행을 정의하는 태그
<th> Table Head Cell 테이블의 제목 셀을 정의하는 테그
<td> Table Data 각각의 데이터 셀을 정의하는 태그
<tfoot> Table Foot 테이블의 footer 행을 정의하는 태그

Table Attributes

Attribute Name Discription
colspan Column Span 이 셀이 몇 개의 열을 병합할 지 정의하는 속성(수평 방향 병합)
rowspan Row Span 이 셀이 몇 개의 행을 병합할 지 정의하는 속성 (수직 방향 병합)
